# ADR-001: Padrão de Autenticação JWT

## Status
Aceito

## Contexto
O sistema precisa autenticar usuários de forma stateless para escalar horizontalmente.

## Decisão
Adotar JWT (JSON Web Tokens) com ECDSA para assinatura, com tokens de acesso curtos (15 min) e Refresh Tokens rotativos guardados em HTTP-Only Cookies.

## Consequências
### Positivas
- Facilidade de escala (stateless).
- Segurança contra XSS (cookies http-only).
### Negativas
- Dificuldade técnica de invalidar tokens antes da expiração.
