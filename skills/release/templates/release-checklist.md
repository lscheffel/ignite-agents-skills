# Release Checklist Template

## Pré-Release

- [ ] CHANGELOG.md atualizado
- [ ] Versão bumpada
- [ ] Testes passam
- [ ] Lint passa
- [ ] Build funciona
- [ ] Security scan

## Publicação

- [ ] Tag criada

- [ ] Docker push
- [ ] GitHub Release

## Pós-Release

- [ ] Usuários notificados
- [ ] Documentação atualizada
- [ ] Versão para dev incrementada
