Decision Rules

Automatic rejection:

  • critical vulnerability
  • secret exposure
  • authentication failure
  • undefined behavior
  • missing audit

Automatic escalation:

  • consensus score < 0.75
  • conflicting findings
  • reviewer timeout

Automatic approval:

  • consensus score ≥ 0.90
  • no blocking findings
